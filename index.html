This version solves arbitrary axial pin-jointed arrangements from supports, nodal loads and load boxes. It also adds workplane drawing, snapping, basic inferencing, simple sketch aids, representative beam and load geometry, and custom rigging libraries. It is not yet a full rigid-body spreader-beam / shackle-side-load / CAD-constraint engine.
